Calculator -- Main Page


The main page for my calculator program.


This is a simple(ish) calculator program coded in C# without the math library.

All the source code is in this GitHub repo.

Works on any operating system, precompiled binaries can be found here.

Coded originally in Visual Studio, then in Vim, now in VSCode, run with dotnet run.

Current features:

  • Addition
  • Subtraction
  • Multiplication
  • Division
  • Square roots
  • Finding the zeros of real quadratic equations

    • Future plans:

  • Ability to evaluate normal equations (i.e. 2*2)
  • Ability to evaluate equations with more than one operation
  • Storing answer in Ans, so that you can perform calculations on the previous answer
  • Variables
  • Solving for x or other variables
  • Add mathematical constants
